# 🇮🇳 Shortest Path Finder (India Map)

This is a Django-based web application that finds the shortest path between two cities in India using Dijkstra’s Algorithm.
It also shows the path on an interactive map using Leaflet.js.

---

## 🚀 Features

* Select source and destination cities
* Find shortest path using Dijkstra’s algorithm
* Show the path on the map
* Simple and fast execution

---

## 🛠️ Tech Stack

* Backend: Django (Python)
* Frontend: HTML, CSS, JavaScript
* Map: Leaflet.js
* Algorithm: Dijkstra

## 🧠 How it works

* Each city is treated as a node
* Distance between cities is considered as edges
* Dijkstra’s algorithm is used to calculate the shortest path
* The result is displayed on the map
